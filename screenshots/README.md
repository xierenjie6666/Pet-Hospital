## 📸 截图要求

### 格式要求
- **文件格式**: PNG格式
- **分辨率**: 建议 1920x1080 或更高
- **文件大小**: 单张不超过2MB

### 内容要求
- 截图清晰，文字可读
- 尽量展示完整界面
- 敏感信息需打码处理（如真实手机号、邮箱等）
- 可以添加适当的标注和说明

### 命名规范
- 使用小写字母
- 单词间用连字符(-)连接
- 遵循上述文件名列表

## 🛠️ 截图工具推荐

### Windows
- **Snipaste** (推荐): 功能强大，支持标注
- **Win + Shift + S**: Windows自带截图工具
- **PicPick**: 专业的截图编辑工具

### Android
- **电源键 + 音量减**: 系统截图
- **截图后编辑**: 添加标注

### 浏览器
- **F12 开发者工具**: 调整为移动端视图
- **响应式设计模式**: 模拟不同设备

## 📝 添加截图后的操作

1. 将截图文件保存到此文件夹
2. 确保文件名与README.md中引用的路径一致
3. 检查README.md中的图片是否正常显示
4. 提交代码到Git仓库

## 💡 提示

- 截图应真实反映系统功能
- 保持界面整洁，避免显示过多无关内容
- 可以适当添加箭头、文字标注说明关键功能
- 建议在白天或明亮环境下截图，避免模糊
