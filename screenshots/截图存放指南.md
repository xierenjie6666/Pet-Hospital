## 🛠️ 截图工具推荐

### Windows平台推荐工具

#### 1. **Snipaste**（强烈推荐）
- **特点**: 功能强大，支持标注、贴图、取色
- **下载**: https://www.snipaste.com/
- **使用方法**:
  ```
  1. 按 F1 开始截图
  2. 选择截图区域
  3. 按 Ctrl+S 保存到指定位置
  4. 或按 Ctrl+C 复制到剪贴板
  ```

#### 2. **Windows自带截图工具**
- **快捷键**: Win + Shift + S
- **优点**: 无需安装，系统自带
- **使用方法**:
  ```
  1. 按 Win + Shift + S
  2. 选择截图模式（矩形/窗口/全屏）
  3. 截图自动保存到剪贴板
  4. 打开画图工具粘贴并保存
  ```

#### 3. **ShareX**（开源免费）
- **特点**: 功能丰富，支持多种截图方式
- **下载**: https://getsharex.com/
- **优点**: 支持自动保存、上传、标注

### 浏览器截图工具

#### 1. **Chrome开发者工具**
- **快捷键**: F12 → Ctrl + Shift + P → 输入 "screenshot"
- **优点**: 可以截取整个页面或特定元素
- **使用方法**:
  ```
  1. 打开网页
  2. 按 F12 打开开发者工具
  3. 按 Ctrl + Shift + P 打开命令面板
  4. 输入 "Capture full size screenshot" 或 "Capture screenshot"
  ```

#### 2. **浏览器扩展插件**
- **Awesome Screenshot**: 支持全页面截图和标注
- **Lightshot**: 快速截图和分享
- **FireShot**: 支持多种截图格式

### Android设备截图方法

#### 1. **系统截图**
```
方法1: 电源键 + 音量减键（同时按）
方法2: 手势截图（三指下滑，需在设置中开启）
方法3: 快捷截图（下拉通知栏点击截图按钮）
```

#### 2. **Android Studio截图**
```
1. 连接Android设备或启动模拟器
2. 打开Logcat窗口
3. 点击左侧的相机图标
4. 选择保存位置和格式
```

#### 3. **第三方截图APP**
- **Screenshot Easy**: 简单易用，多种截图方式
- **Super Screenshot**: 支持长截图和标注

### Postman截图方法

#### 1. **请求界面截图**
```
1. 打开Postman
2. 配置API请求
3. 使用Snipaste或其他工具截取整个Postman窗口
4. 确保请求参数和响应结果都可见
```

#### 2. **响应结果截图**
```
1. 发送请求
2. 查看响应结果
3. 截取响应部分（包含状态码、返回数据）
4. 确保关键信息清晰可见
```

## 💾 截图文件管理

### 1. 文件命名规范

**统一命名规则：**
```
平台-功能页面.png

示例：
api-register.png       # API + 注册功能
web-login.png          # Web平台 + 登录页面
android-user-home.png  # Android平台 + 用户主页
```

### 2. 文件格式要求

**推荐格式：**
- **PNG**: 最佳选择，清晰度高，支持透明
- **JPG**: 可选，文件较小，但可能有质量损失

**分辨率要求：**
```
最小: 1280 x 720
推荐: 1920 x 1080
最佳: 2560 x 1440
```

### 3. 文件大小控制

**压缩方法：**
```
1. 使用在线压缩工具：
   - https://tinypng.com/
   - https://squoosh.app/

2. 使用图片编辑软件：
   - Photoshop → 导出为Web格式
   - GIMP → 导出时调整质量

3. 控制单张图片大小在 2MB 以内
```
